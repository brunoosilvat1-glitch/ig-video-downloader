# Baixador local de vídeo do Instagram

App local, sem API externa. Cole um link público de Reel/post do Instagram, carregue o vídeo, veja a descrição pública e baixe o arquivo.

## Rodar

```bash
npm start
```

Abra:

```text
http://127.0.0.1:5176
```

## Observações

- O app não melhora a qualidade real do vídeo. Ele baixa a melhor URL pública que o Instagram expõe na página.
- Posts privados, stories, carrosséis, links que exigem login ou páginas que o Instagram bloquear podem não carregar.
- Use apenas para vídeos seus ou quando você tiver permissão para baixar.
